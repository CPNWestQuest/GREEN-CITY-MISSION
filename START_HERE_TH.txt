GREEN CITY MISSION — คนเซ็นทรัล เปลี่ยนเมือง เปลี่ยนโลก
============================================================

ไฟล์ในแพ็กเกจ
- index.html      หน้าเว็บเกม
- style.css       รูปแบบ แอนิเมชัน และ Responsive Design
- app.js          ระบบเกม 10 รอบ โครงการ เหตุการณ์ คะแนน และ Google Sheets
- config.js       รายชื่อสาขาและ URL ของ Apps Script
- Code.gs         Backend สำหรับ Google Sheets
- VERSION.txt     หมายเลขเวอร์ชัน

คุณสมบัติของเกม
- ลงทะเบียน ชื่อ นามสกุล รหัสพนักงาน สาขา เบอร์โทร The1
- มีตัวเลือก “อื่น ๆ” และช่องกรอกสาขาเพิ่มเติม
- งบเริ่มต้น 100 Green Coins
- เล่น 10 รอบ เลือก 1 จาก 3 โครงการในแต่ละรอบ
- มีโครงการเพื่อสิ่งแวดล้อมและตัวเลือกที่ต้องคิดให้รอบคอบ
- มีเหตุการณ์สุ่มระหว่างเกม เช่น คลื่นความร้อน ฝนตกหนัก เทศกาล และมลพิษ
- ตัวชี้วัด 5 ค่า: ความสุข ขยะ มลพิษ พื้นที่สีเขียว พลังงานสะอาด
- ภาพเมือง SVG แบบแอนิเมชัน เปลี่ยนตามโครงการที่เลือก
- รองรับมือถือ แท็บเล็ต และคอมพิวเตอร์
- เล่นได้แม้ยังไม่เชื่อม Google Sheets
- มีระบบคิวเก็บข้อมูลไว้ส่งใหม่เมื่อออนไลน์

อัปเกมขึ้น GitHub Pages
1. แตกไฟล์ ZIP
2. เข้า Repository ที่ต้องการใช้
3. Add file > Upload files
4. ลากไฟล์ทั้งหมดในโฟลเดอร์ขึ้นไปที่หน้าแรกของ Repository
   ต้องเห็น index.html, app.js, style.css, config.js อยู่ที่ระดับบนสุด
5. Commit changes
6. Settings > Pages > Deploy from a branch > main / root
7. รอ Actions ขึ้นเครื่องหมายถูกสีเขียว

ตั้งค่า Google Sheets
1. เปิด Google Sheets ที่ต้องการใช้
2. Extensions > Apps Script
3. ลบโค้ดเดิมและวางเนื้อหาจาก Code.gs
4. ตรวจสอบ SPREADSHEET_ID ใน Code.gs
5. เลือกฟังก์ชัน setupSheets แล้วกด Run หนึ่งครั้ง
6. อนุญาตสิทธิ์ตามขั้นตอน
7. Deploy > New deployment > Web app
8. Execute as: Me
9. Who has access: Anyone
10. Deploy และคัดลอก URL ที่ลงท้ายด้วย /exec
11. เปิด config.js แล้วใส่ URL ใน API_URL
12. อัป config.js ขึ้น GitHub ทับไฟล์เดิม

ชีตที่จะถูกสร้างอัตโนมัติ
- GreenCity_Players
- GreenCity_Choices
- GreenCity_Results
- GreenCity_Errors

การทดสอบ
- เปิดเกมในหน้าต่างไม่ระบุตัวตน
- ลงทะเบียนและเล่นให้ครบ 10 รอบ
- ตรวจสอบว่า Google Sheets มีข้อมูลในทั้ง 3 ชีตหลัก

หมายเหตุ
เกมสามารถเล่นแบบออฟไลน์ได้ทันที หาก API_URL ยังว่าง ข้อมูลจะไม่ส่งเข้า Google Sheets
